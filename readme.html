BMagicSEO PHP Class - made by B3aT (BTooLs.eu)

The main function of the class : convert any kind of text (html,php,utf8,etc) to clean,safe,friendly SEO tags and links. Also calculate the Keyword density to improve your SEO.

Version : 0.9 2011.09.2

First step

If your framework doesn`t autoload the class, include the class file :

require ('bmagicseo.class.php');

Instantiate the class

$bseo = new BMagicSeo();

Make your own options, and setup the unwanted prepositions

$bseo->set_options($my_setings);//send your options.

Make your links and meta tags.

$bseo->seo_magic_link('<h1>My &New #25 inches product!</h1>',20);//result : my-new-25-inches-product

Create a link with the product id included (45)

$bseo->seo_magic_link('<h1>My &New #25 inches product!</h1>',20,45,false,'--');//result : my_new_25_inches_product--45

For more examples regarding the other functions see example.php (source + output)
